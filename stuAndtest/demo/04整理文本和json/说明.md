- 1.初始文件结构: ==一个角色一个文件夹==
  ```js
  - character_1
    - photos 
      - a.png
      - b.jpeg ...
    - c_cover.png
    - d_avatar.jpg 
    - character.txt
  ```
  > 支持[".jpg", ".jpeg", ".png", ".txt"]
- 2.生成结果： 
  ```js
    + character_avacov/  cover和avatar图片集合
    + character_photos/  photos图片结合
    + character_result.json  可插入character集合的数据
  ```
- 3.脚本文件功能：
  ```js
    - parseTxtToJson.js: 把对应的txt文件转化为json数据
    - handleJson.js: 整理数据生成json文件和分类图片
  ```
- 4.执行命令: 在`handleJson.js`的父文件夹终端下执行  
  ```bash
    node handleJson.js
  ```
